当社では、シーケンス設計・機構設計・回路設計・アートワーク設計・ソフト開発・を行う開発部・技術部と基板実装・配線・組立を行う生産部があります。社内設計したものを同時に社内加工してしているので製造しやすさを考慮した効果的な設計・組立が行われています。
社内で連携しているので外部に情報が漏れることもありません。

生産部

生産部1課では主に制御盤製作や電線加工及び装置の組立を、部品調達から出荷検査まで行っております。

生産部2課ではプリント基板の実装やそれに付随する装置の組立を、部品調達から出荷検査まで行っております。

開発部

電気回路設計からプリント基板の電子回路設計およびアートワーク設計、組込みソフトウェア(マイコン・CPLD)設計、PCアプリケーションのソフトウェア(C++/C#)設計、ラック・筐体・治具などの構造設計まで幅広く行っています。

民需
産業向けの電子部品の計測・検査装置の設計開発
特機
主に陸自・海自向けの防衛機器電装品、及びそれらの試験器の設計開発などを行っています。

技術部

生産設備の設計を行っています。

設計業務の流れ

  1. 1.客先との仕様取り決め、見積資料作成。
  2. 2.受注後、製作仕様書を作成し、客先確認の上、回路図、部品表、板金図、ケーブル加工図を作成します。
  3. 3.生産部で装置組立、配線終了後、技術部で機械調整デバック後、出荷検査を行い、お客様立ち合いを受けます。
  4. 4.出荷後、納入先に検収を受け、提出書類を提出して終了いたします。
