あなたはどのような会社で働きたいですか？

わたしたち株式会社イシメックスは、完全公休＆定時帰社を基本としてますので、ひっ迫期以外は残業することはほとんど無いです。それは、「プライベートを大事にするから仕事も頑張れる。」の発想からくるものかもしれません。

「原則、転勤は発生しません！安心して人生設計が構築できます。人情が厚い上司や先輩が多いので、気疲れや人間関係で悩むことが無く、働きやすい環境です。 夏冬作業服・作業靴は支給致します。 作業場含めてエアコン完備！最適な環境で仕事に打ち込めます。」

「土曜日曜祝日の公休、夏季休暇・年末年始休暇・GW等の大型連休等に有給を織り交ぜてしっかり遊んで休息して、毎日を定時で帰宅する」。このようなライフスタイルが充実できる「イシメックス」で一緒に成長をしませんか？
